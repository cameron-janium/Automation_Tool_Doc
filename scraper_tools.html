scraper_tools.find_element(parent_element, target_element: str)¶

Used to find elements.

Uses the BeautifulSoup syntax to find an element but adds a check if the parent element is null. This prevents the program from erroring out if the parent is of None type. Turns the BS4 find statement into a string and gets the desired element.

Parameters:

  • parent_element (bs4.element.Tag) – This is the parent element of the element you are finding.

  • target_element (str) – This is the BS4 find function as a string. (E.g. “find(‘ul’, class_=’msg-s-message-list-content’)”)

Returns:

Will either return None if parent or targeted element doesn’t exist, or will return the targeted element.

Return type:

bs4.element.Tag
