utility module¶

utility.bezier_curve(points, num_samples)¶

Generates intermediate X Y cordinates to a ending destination.

Used in the move_mouse function to produce randomized mouse movements. This function produces random X Y cordinates that will give the mouse a random non-straight mouse movement. In other words, this function makes the mouse move in a random curved motion.

Parameters:

  • points (List) – These are the points produced in the get_intermediate_points function.

  • num_samples (int) – The desired number of intermediate X Y cordinates to produce.

Returns:

Two seperate arrays for the X and Y cordinates.

Return type:

List

utility.get_intermediate_points(ending_cords: tuple)¶

Generates 2 randomized points to be used in the bezier_curve function.

Produces 2 random points that are somewhere inbetween the current mouse position and the desired mouse cordinates.

Parameters:

ending_cords (tuple) – These are the destination cordinates for the mouse

Returns:

A list of 4 cordinate tuples. The current mouse position, the 2 randomized points, and the destination cordinates.

list

utility.wind_mouse(start_x, start_y, dest_x, dest_y, G_0=9, W_0=3, M_0=35, D_0=12)¶

Generates more randomized mouse movements to be used in the move_mouse function.

This function attempts to generate more human-like movements by creating randomness in mouse movements. See ‘https://ben.land/post/2021/04/25/windmouse-human-mouse-movement/’ for more info.

WindMouse algorithm. Calls the move_mouse kwarg with each new step. Released under the terms of the GPLv3 license.

Parameters:

  • start_x (int) – current mouse X position

  • start_y (int) – current mouse Y position

  • dest_x (int) – destination mouse X position

  • dest_y (int) – destination mouse Y position

  • G_0 (int) – magnitude of the gravitational fornce

  • W_0 (int) – magnitude of the wind force fluctuations

  • M_0 (int) – maximum step size (velocity clip threshold)

  • D_0 (int) – distance where wind behavior changes from random to damped
